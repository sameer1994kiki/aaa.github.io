node.js+hexo+github 搭建静态博客

2016-12-01

1.安装nodejs git
2.在命令工具中安装hexo ： npm install -g hexo（我习惯用node）
3.执行init命令初始化hexo： hexo init
4.新建文章命令： hexo new”postName” (hexo new page”pageName” #新建页面)
5.生成静态页面至public目录 hexo generate
6.开启预览访问端口 hexo server （默认端口4000，’ctrl + c’关闭server）
7.将.deploy目录部署到GitHub hexo deploy （我的用该方法总是各种报错，配置了环境变量，然后又报其他错误，所以我使用github的可视化界面直接将public文件夹copy到github个人主页中)
8.修改页面内容只需在theme中的config中修改即可

报错总结：
1.Tags 要另起一行，以- name 的格式编写 //-后要有一个空格
或 tags: [tag1,tag2,tag3] 冒号后有一个空格
否则会报错 process failed

2.更改主题：
$ git clone https://github.com/litten/hexo-theme-yilia.git themes/human （主题地址，放到themes下的human
）
cd themes/human

git pull

把_config.yml文件的主题改为human

hexo geneate
然后重新启动hexo server
